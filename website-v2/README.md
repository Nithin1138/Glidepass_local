# 🚀 LANpad Startup-Grade Landing Page (v2)

This is the high-fidelity version of the LANpad landing page, built with the industry-standard tech stack for premium startups.

## 🛠 Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Typography:** Outfit (Headings), Inter (Body), JetBrains Mono (Code)

## 🚀 How to Run Locally

```bash
cd website-v2
npm install
npm run dev
```

Visit `http://localhost:3000` to see the site.

## 🏗 Build for Production

```bash
npm run build
```

The optimized output will be in the `.next` folder, ready for deployment to Vercel.

## ✨ Features
- **Silicon Valley Aesthetic:** Dark mode, glassmorphism, and tan/brown accent palette.
- **Dynamic Visualization:** Interactive "Phone-to-Laptop" sync demo with smooth character-by-character typing and satellite motion.
- **Responsive Bento Layout:** Features showcased in a modern grid system.
- **Performance Optimized:** Built with Next.js for sub-second load times and perfect SEO scores.
