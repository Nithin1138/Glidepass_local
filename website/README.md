# 🌐 LANpad Landing Page

A premium, high-performance landing page for LANpad, built with Vite and Vanilla JS.

## 🚀 Quick Start

To preview the website locally:

```bash
cd website
npx vite
```

## 🏗️ Building for Production

To generate the optimized static files for hosting (Vercel, Netlify, GitHub Pages):

```bash
cd website
npx vite build
```

The output will be in the `dist/` folder.

## ✨ Features
- **Hero Section:** Dynamic visualization of phone-to-laptop syncing.
- **Product Page:** Highlight core value propositions (Flash Paste, Realistic Typing).
- **Documentation:** Sidebar-driven guides for setup and API.
- **Downloads Hub:** Quick access to cross-platform binaries.
- **Onboarding:** Step-by-step interactive guide for new users.
